¿Qué son las listas en HTML?

Las listas en HTML existen desde las primeras versiones del lenguaje (1995). Sirven para agrupar elementos relacionados de forma clara, igual que una lista escrita en papel. Se usan en todas partes: menús de navegación, pasos de una receta, resultados de búsqueda, características de un producto.

HTML ofrece dos tipos: <ol> (ordered list) cuando el orden sí importa, y <ul> (unordered list) cuando el orden no importa. Los dos usan <li> para cada elemento.
