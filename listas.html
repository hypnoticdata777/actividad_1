¿Qué es una lista en HTML?

Imagina que vas al supermercado sin una lista... acabas comprando helado y olvidando la leche. Las listas existen para agrupar cosas relacionadas en orden. HTML tiene dos tipos principales: las que sí importa el orden y las que no importa el orden.

🔢

Lista Ordenada <ol>

Úsala cuando el orden importa, como una receta o instrucciones paso a paso. El navegador pone los números (o letras, o romanos) solo.

Ejemplo real:

  1. Prender la computadora
  2. Abrir el editor de código
  3. Escribir HTML
  4. Ver el resultado en el navegador

Tip: el atributo type="A" cambia los números por letras, y type="I" los cambia por romanos.

🔵

Lista No Ordenada <ul>

Úsala cuando el orden no importa, como una lista de ingredientes o las páginas de un menú de navegación. En vez de números usa viñetas (●, ○, ■).

Ejemplo real:

  • Arroz
  • Frijoles
  • Aguacate
  • Tortillas

Tip: con CSS puedes cambiar el estilo de la viñeta usando list-style-type: circle o square.

⚡

Resumen rápido

¿Cuándo usarla? Etiqueta Viñeta
El orden SÍ importa (pasos, ranking) <ol> 1, 2, 3… / A, B… / I, II…
El orden NO importa (colección, menú) <ul> ● / ○ / ■
