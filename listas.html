Listas en HTML

📋

¿Qué es una lista en HTML?

Imagina que vas al supermercado sin una lista... acabas comprando helado y olvidando la leche. Las listas en HTML sirven para agrupar elementos relacionados de forma ordenada y clara.

HTML tiene dos tipos principales de lista:

  • Ordenada <ol> — cuando el orden SÍ importa
  • No ordenada <ul> — cuando el orden NO importa

Dato extra: las listas se pueden anidar, es decir, poner una lista dentro de otro <li> para crear submenús o subniveles.

🔢

Lista Ordenada <ol>

Usa <ol> cuando el orden importa: instrucciones paso a paso, rankings, recetas. El navegador pone la numeración automáticamente, tú solo pones los <li>.

Ejemplo real — pasos para programar:

  1. Prender la computadora
  2. Abrir el editor de código
  3. Escribir HTML
  4. Ver el resultado en el navegador

Con números type por defecto

Si no escribes ningún atributo type, la lista usa números automáticamente: 1, 2, 3… Es el tipo más común para instrucciones o pasos.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Con letras type="A"

El atributo type="A" cambia los números por letras mayúsculas: A, B, C… Útil para exámenes, opciones de respuesta o secciones de un documento formal.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Con romanos type="I"

El atributo type="I" usa numeración romana en mayúscula: I, II, III… Se ve elegante en índices, capítulos o documentos con estilo clásico.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Otros valores de type: "a" (letras minúsculas), "i" (romanos en minúscula). También existe el atributo start="5" para empezar la numeración desde otro número.

🔵

Lista No Ordenada <ul>

Usa <ul> cuando el orden no importa: listas de ingredientes, características de un producto, menús de navegación. En vez de números usa viñetas (●, ○, ■).

Ejemplo real — lista del súper:

  • Arroz
  • Frijoles
  • Aguacate
  • Tortillas

Con disc por defecto

list-style-type: disc muestra un círculo relleno ●. Es el valor por defecto si no escribes nada. Se aplica con CSS o con el atributo HTML style directo en la etiqueta.

  • Primer elemento
  • Segundo elemento
  • Tercer elemento

Con circle list-style-type: circle

list-style-type: circle muestra un círculo vacío ○. Se usa mucho en sublistas (listas anidadas dentro de otra lista) para distinguir el nivel de profundidad visualmente.

  • Primer elemento
  • Segundo elemento
  • Tercer elemento

Con square list-style-type: square

list-style-type: square muestra un cuadrado ■. Da un aspecto más moderno y menos redondeado. Ideal para listas en diseños con estética geométrica o minimalista.

  • Primer elemento
  • Segundo elemento
  • Tercer elemento

Con CSS moderno también puedes usar emojis o imágenes como viñeta con la propiedad list-style-image, o quitarlas por completo con list-style: none (como en el menú de navegación de este mismo sitio).
